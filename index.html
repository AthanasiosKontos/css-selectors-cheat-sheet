HTML & CSS Cheat Sheet

HTML Overview

Tag Example Description
<h1> to <h6> <h1>Heading</h1> Defines HTML headings.
<a> <a href="url">Link</a> Defines a hyperlink.
<p> <p>Paragraph</p> Defines a paragraph.
<ul> <ul><li>Item</li></ul> Defines an unordered list.
<table> <table><tr><th>Header</th></tr><tr><td>Data</td></tr></table> Defines a table.

CSS Selectors

Selector Example Description
.class .example { color: black; } Selects all elements with class="example".
#id #unique { color: black; } Selects the element with id="unique".
* * { margin: 0; } Selects all elements.
element h1 { font-size: 24px; } Selects all <h1> elements.
:hover a:hover { color: red; } Applies to links on mouse-over.

Specificity

Specificity determines which CSS rule is applied by the browsers when multiple rules could apply to the same element. It is calculated based on the number of ID selectors, class selectors, and element selectors.
